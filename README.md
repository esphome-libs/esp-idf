# esp-idf — Stripped Mirror

Automated mirror of [espressif/esp-idf](https://github.com/espressif/esp-idf) releases with the `.git` directory removed from the release archives.

## Why?

The official esp-idf release zips include a `.git` directory with the full repository history, making them much larger than needed for build-only use cases.

## Download

Same filename as upstream, just swap the repo prefix:

```
https://github.com/esphome-libs/esphome-libs/releases/download/<TAG>/esp-idf-<TAG>.zip
https://github.com/esphome-libs/esphome-libs/releases/download/<TAG>/esp-idf-<TAG>.tar.gz
https://github.com/esphome-libs/esphome-libs/releases/download/<TAG>/esp-idf-<TAG>.tar.xz
```
